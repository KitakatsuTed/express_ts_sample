## 環境構築

```sh
npm install
npm run dev
```

## サーバー立ち上げ
```sh
npm run client # クライアント側サーバー(webpack)起動 :8080
npm run dev # バックエンド側サーバー起動 :3000
```

localhost:8080で開発進めるすすめる
